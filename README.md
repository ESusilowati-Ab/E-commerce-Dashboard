# 🤖 DataMind AI – AI Data Analyst Copilot

An AI-powered data analytics web application built with React, TypeScript, and Vite. DataMind AI helps users clean, analyze, visualize, and explore datasets through an intuitive interface with AI-assisted insights.

---

## 🚀 Features

- 📂 Import and manage datasets
- 🧹 Data cleaning and preprocessing
- 📊 Interactive dashboard
- 📈 Statistical analysis
- 📉 Data visualization
- 🔍 SQL Explorer
- 🤖 AI Chat Assistant for data analysis
- 📋 Automated reports
- 📤 Export processed data
- ⚙️ Settings and configuration

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ESusilowati-Ab/E-commerce-Dashboard.git
```

Navigate to the project

```bash
cd E-commerce-Dashboard
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
